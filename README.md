# 📖 Meu Leitor de PDF

App leitor de PDF com voz, biblioteca e progresso salvo.

## Como publicar na Vercel (gratuito)

1. Faça upload destes arquivos no GitHub
2. Acesse vercel.com e conecte seu GitHub
3. Importe o repositório — a Vercel detecta React automaticamente
4. Clique Deploy — pronto!

## Estrutura de arquivos

```
meu-leitor-pdf/
├── package.json
├── public/
│   ├── index.html
│   └── manifest.json
└── src/
    ├── index.js
    └── App.js   ← código principal
```
